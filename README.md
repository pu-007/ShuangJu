## 核心功能

1. 主界面：日历
    1. 显示年月日、农历、节日
    2. 强制横屏显示，支持全屏显示
    3. 每日随机一个来自电视剧管理系统的电视剧，背景图片为电视剧剧照，且在日期下显示电视剧的台词。先读取剧照的台词，如果没有则随机选择一句该电视剧台词

2. 子功能一：电视剧相册与台词以及进度管理
    1. 用于电视剧的管理与回顾，为日历背景图的数据源
    2. 内置一部分电视剧，也支持用户自己添加，可以读取多个目录，将这些目录作为数据源目录
    3. 存储格式为读取数据源目录，目录格式如下
        1. 一级目录中有很多电视剧的文件夹，每个文件夹名称就是这个电视剧的名称
        2. 二级目录，电视剧文件夹中，有 init.json 记录电视剧元信息，有观看进度，名称，经典台词列表（用于日历读取），个人想法，还有目录内剧照图片内联台词，用相对路径标注剧照对应的台词，用于日历优先读取
        3. 二级目录中存储这部电视剧的所有剧照，每个剧照都有唯一编号，自动格式化名称为日期+唯一识别码
    4. 在日历界面点击该功能按钮就可以显示这个管理系统，默认是当前电视剧，就是今日随机到的电视剧，有一个显示并且编辑这个电视剧观看进度的空间，还有编辑台词列表和个人想法列表，上传和管理图片的空间，可以修改管理对应的数据目录
    5. 还有另一个子项是管理所有电视剧对象，列出所有的来自各个数据源的电视剧对象，点击可进行上述的管理，还可以创建新的电视剧
    6. 在电视剧界面，支持跳转到播放电视剧的网页/app，要支持多个数据源，支持自定义，使用 url-schema 打开应用，或者https打开网页，用电视剧名称作为参数，在代码内内置数据源列表，预留相关位置
    7. 支持跳转 TMDB 界面搜索，查看电视剧信息
    8. 要支持收藏最喜欢的电视剧，要有一个界面显示，同时支持搜索，记录到 init.json 中

3. 子功能二：简约日志笔记
    1. 每天可以创建新的一页笔记，用markdown格式存储，附带一个简单的编辑器，最好使用现成组件，有一个简单的toolbar可以加粗添加图片等操作
    2. 支持快捷添加 TODO，并且自动显示所有的每天的 TODO

4. 子功能三：搜索，支持搜索已经有的电视剧名称，台词，个人想法，和笔记，支持筛选

5. 子功能四：菜单内添加一个`爽`按钮，点击后可以播放一段生日祝福视频，强制横屏播放，自动打开音量，视频文件集成在安装包内，预留视频文件的路径

## 注意事项与要求

1. 在代码中，为内置数据源文件夹，生日祝福视频，留下注释，告诉我应该在哪里放置

2. 要确保 UI 美观，功能布局合理

3. init.json 格式如下

```json
{
    "name": "星汉灿烂",
    "progress": {
        "current": 0,
        "total": 60
    },
    "favorite": true,
    "notes": [
        "这是一部很好的电视剧"
    ],
    "lines": [
        "xxxxxxx",
        "xxxxxx",
    ],
    "inline_lines": {
        "2025-03-23-xxxxx.jpg": "xxxxxxxxx",
    }
}
```

4. 数据源势力，默认数据源，用 name 表示电视剧名称，点击跳转时候自动替换

```json
[
    {"爱壹帆": "https://www.yfsp.tv/search/{name}"}
]
```

5. 要确保能正常显示大量图片，还有功能的完备性，健全性
